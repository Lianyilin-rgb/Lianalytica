# 音乐创作者使用指南 - 连析工坊(Lianalytica)

## 欢迎使用🎵
本指南将帮助音乐人、作曲家、歌词创作者、音频制作人等专业人士使用WebUI进行音频分析、音乐创作、歌词生成和音乐知识获取。

## 一、安装步骤

### 1.1 下载适合的版本
对于音乐创作者，推荐下载以下版本：
- `OllamaWebUI_GPU.exe`（GPU加速版本，处理音频文件速度更快）
- 若您的电脑没有独立显卡，可选择`OllamaWebUI_CPU.exe`

### 1.2 安装与启动
1. 双击下载好的启动器图标
2. 首次使用会自动安装 Ollama（请耐心等待，可能需要几分钟）
3. 安装完成后，浏览器会自动打开 Ollama WebUI 界面

## 二、音频分析工作流程

### 2.1 音频文件上传
1. **登录界面**：打开软件后，输入任意用户名即可登录
2. **进入音频界面**：点击左侧导航栏的「音频分析」按钮
3. **上传音频文件**：
   - 点击界面中央的「选择文件」按钮
   - 支持常见音频格式：MP3、WAV、FLAC、OGG 等
   - 建议单个文件大小不超过 100MB

### 2.2 音频分析功能

#### 2.2.1 频谱分析
1. **选择分析类型**：在「分析类型」下拉菜单中选择「频谱分析」
2. **调整参数**：
   - **分析深度**：默认为 0.8，数值越高分析越详细
   - **频谱分辨率**：选择适合的分辨率（低/中/高）
3. **开始分析**：点击「开始分析」按钮
4. **查看结果**：
   - 实时频谱图
   - 频率分布直方图
   - 音频能量变化曲线
   - 关键频率点标注

#### 2.2.2 节拍与调性分析
1. **选择分析类型**：在「分析类型」下拉菜单中选择「节拍与调性分析」
2. **开始分析**：点击「开始分析」按钮
3. **查看结果**：
   - BPM（每分钟节拍数）
   - 音乐调性（如 C 大调、A 小调）
   - 节拍强度分布
   - 调性变化点标注

#### 2.2.3 音频内容识别
1. **选择分析类型**：在「分析类型」下拉菜单中选择「音频内容识别」
2. **开始分析**：点击「开始分析」按钮
3. **查看结果**：
   - 乐器识别结果
   - 人声与伴奏分离建议
   - 音频质量评估
   - 潜在改进建议

## 三、音乐创作功能

### 3.1 旋律生成
1. **进入创作界面**：点击左侧导航栏的「对话」按钮
2. **生成旋律**：
   - 在输入框中描述您想要的旋律风格，例如：
     - "请生成一段轻快的流行音乐旋律，BPM 120，C大调"
     - "帮我创作一段悲伤的钢琴旋律，适合电影配乐"
   - 点击「发送」按钮
3. **查看结果**：
   - 旋律的音符序列（可直接用于音乐制作软件）
   - 和弦进行建议
   - 演奏技巧提示

### 3.2 歌词创作
1. **进入创作界面**：点击左侧导航栏的「对话」按钮
2. **生成歌词**：
   - 在输入框中描述您想要的歌词主题和风格，例如：
     - "请帮我写一首关于青春的流行歌曲歌词，主歌+副歌结构"
     - "创作一段古风歌词，主题是思乡"
   - 点击「发送」按钮
3. **查看结果**：
   - 完整的歌词内容
   - 押韵建议
   - 情感表达优化建议

### 3.3 编曲建议
1. **进入创作界面**：点击左侧导航栏的「对话」按钮
2. **获取编曲建议**：
   - 在输入框中描述您的音乐需求，例如：
     - "我有一段吉他旋律，BPM 90，A小调，请给我编曲建议"
     - "帮我为这首流行歌曲添加合适的鼓点和贝斯线"
   - 点击「发送」按钮
3. **查看结果**：
   - 详细的编曲方案
   - 乐器选择建议
   - 和声进行优化
   - 混音技巧提示

## 五、高级功能

### 5.1 批量音频分析
1. **进入音频界面**：点击左侧导航栏的「音频分析」按钮
2. **选择批量分析**：
   - 点击「批量分析」按钮
   - 选择多个音频文件或整个文件夹
3. **设置分析参数**：选择需要的分析类型和参数
4. **开始批量分析**：点击「开始分析」按钮
5. **导出结果**：分析完成后，点击「导出报告」按钮，可导出为Excel或PDF格式

### 5.2 音乐风格迁移
1. **进入创作界面**：点击左侧导航栏的「对话」按钮
2. **描述风格迁移需求**：
   - "请将这段古典音乐旋律转换成电子音乐风格"
   - "帮我把这首摇滚歌曲改编成爵士版本"
3. **点击发送**：AI会生成风格迁移后的音乐建议

### 5.3 歌词与旋律匹配
1. **进入创作界面**：点击左侧导航栏的「对话」按钮
2. **提供歌词或旋律**：
   - 输入您的歌词，请求AI生成匹配的旋律
   - 或提供旋律，请求AI创作匹配的歌词
3. **调整匹配度**：根据需要调整歌词与旋律的匹配参数
4. **查看结果**：AI会生成高度匹配的歌词或旋律

## 七、常见问题与解决方案

### 7.1 音频分析速度慢
- 解决方案：降低分析深度参数，或使用GPU加速版本

### 7.2 音频文件无法上传
- 解决方案：检查文件格式是否支持，文件大小是否超过限制

### 7.3 生成的旋律不符合预期
- 解决方案：提供更详细的风格描述，包括BPM、调性、情绪等参数

### 7.4 软件崩溃或无响应
- 解决方案：关闭软件并重新启动，确保电脑有足够的内存和CPU资源
