# 跨平台安装运行测试文档

## 测试目标
验证连析工坊音频处理应用在不同操作系统上的安装和运行稳定性。

## 测试环境
- Windows 10/11 64位
- macOS 10.15+ (Catalina, Big Sur, Monterey, Ventura, Sonoma)
- Linux (Ubuntu 20.04+, Debian 10+, Fedora 34+)

## 测试步骤

### 1. Windows平台测试
1. 双击运行 `audio_processor_windows.exe`
2. 等待应用启动，检查是否有错误提示
3. 测试音频上传和处理功能
4. 验证DAW工程导出功能
5. 检查中文短语显示和翻译功能
6. 运行至少30分钟，观察是否有崩溃或性能问题

### 2. macOS平台测试
1. 双击运行 `audio_processor_macos.dmg`
2. 将应用拖拽到Applications文件夹
3. 打开应用，检查是否有安全提示
4. 测试音频上传和处理功能
5. 验证DAW工程导出功能
6. 检查中文短语显示和翻译功能
7. 运行至少30分钟，观察是否有崩溃或性能问题

### 3. Linux平台测试
1. 解压 `audio_processor_linux.tar.gz`
2. 运行 `./audio_processor_linux`
3. 检查依赖是否安装完整
4. 测试音频上传和处理功能
5. 验证DAW工程导出功能
6. 检查中文短语显示和翻译功能
7. 运行至少30分钟，观察是否有崩溃或性能问题
